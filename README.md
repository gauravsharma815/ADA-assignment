## 🧠 Task 1: Algorithm Growth Observation

Implemented algorithms with different time complexities:

* Constant Time → O(1)
* Linear Time → O(n)
* Quadratic Time → O(n²)
* Logarithmic Time → O(log n)

### 📊 Observation

* O(1) remains constant
* O(n) increases linearly
* O(n²) grows rapidly
* O(log n) grows very slowly

---

## 🔍 Task 2: Linear Search vs Binary Search

Implemented:

* Linear Search
* Binary Search

### 📊 Observation

* Linear Search → O(n)
* Binary Search → O(log n)
* Binary search performs better for large datasets

---

## 🔁 Task 3: Recursion Analysis

Implemented:

* Factorial (Recursive)
* Fibonacci (Recursive)
* Fibonacci (Dynamic Programming)

### 📊 Observation

* Recursive Fibonacci is inefficient (exponential)
* Dynamic Programming reduces time complexity significantly
* Factorial shows linear recursion behavior

---

## 🔄 Task 4: Recurrence Relations

Implemented recurrence functions:

* T(n) = T(n/2) + n → O(n)
* T(n) = 2T(n/2) + n → O(n log n)

### 📊 Observation

* Recursion depth and function calls match theoretical complexity
* Practical execution confirms asymptotic analysis
